# CodeAlpha-WebScraping

## Project Description
This project was developed as part of
the CodeAlpha Data Analytics
Tnternship.

The program uses Python to scrape the
top 10 quotes from the website
https://quotes.toscrape.com.

## Technologies Used
- Python
- Requests
- BeautifulSoup (bs4)

## Features
- Fetches webpage content
- Extracts top 10 quotes
- Displays quotes in the terminal
